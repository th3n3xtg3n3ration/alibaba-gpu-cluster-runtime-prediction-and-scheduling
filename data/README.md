# Alibaba GPU Data Directory

This directory is intended to store the Alibaba PAI GPU workload traces.

## Dataset Source
The data used in this thesis is derived from the **Alibaba PAI GPU Cluster Trace (2022)**.
- **Paper:** "MLaaS in the Wild: Workload Analysis and Scheduling in Large-Scale Heterogeneous GPU Clusters" (NSDI '22).
- **Official Repository:** [Alibaba PAI GPU Trace](https://github.com/alibaba/pai-gpu-trace)

## Structure
- `raw/`: Place the original CSV files here (e.g., `pai_job_table.csv`).
- `processed/`: Intermediate utilization-aware datasets generated by `notebooks/00_prepare_utilization_data.ipynb`.

## Note on Samples
This repository ships with a **100k-row sample** for demonstration and reproducibility of the core logic. To run experiments on the full multi-million row trace, please download the official files and place them in `data/raw/`.
